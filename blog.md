### Build Your First Website with HTML & CSS
**Category:** HTML & CSS | **Year:** 2024 | **Read time:** 4 min

HTML gives structure and CSS gives style. Together they are the foundation of every website.

**Basic HTML Structure:**
```html
<!DOCTYPE html>
<html>
  <head>
    <title>My Website</title>
  </head>
  <body>
    <h1>Hello World!</h1>
  </body>
</html>
```

**Adding CSS Style:**
```css
body {
  background: #0a0a0f;
  font-family: sans-serif;
}
h1 {
  color: #4d96ff;
  text-align: center;
}
```

### How to Host Your Website Free on GitHub Pages
**Category:** Web Dev | **Year:** 2024 | **Read time:** 3 min

GitHub Pages is a completely **free hosting service** by GitHub.

**Step-by-Step Guide:**
1. Create a free account at `github.com`
2. Create a new repository named `yourusername.github.io`
3. Upload your HTML and CSS files
4. Go to Settings → Pages → Enable GitHub Pages
5. Your site is live at `yourusername.github.io`

### Introduction to MySQL — Databases Explained Simply
**Category:** MySQL | **Year:** 2024 | **Read time:** 5 min

MySQL is a **database management system (DBMS)** used to store, organize, and retrieve data.

**Creating Your First Table:**
```sql
CREATE TABLE students (
  id    INT,
  name  VARCHAR(50),
  marks INT
);
```

**Inserting & Viewing Data:**
```sql
INSERT INTO students VALUES (1, 'Shakeel', 95);
SELECT * FROM students;
SELECT name FROM students WHERE marks > 80;
```

### Getting Started with Python & Machine Learning
**Category:** Python & ML | **Year:** 2024 | **Read time:** 6 min

Python is the world's most popular language for **AI and Machine Learning**.

**Your First Python Program:**
```python
print("Hello, World!")
name = "Shakeel"
age = 20
print("My name is", name, "and I am", age, "years old.")
```

**Key Python Libraries for ML:**
- `NumPy` — Mathematical operations on arrays
- `Pandas` — Data handling and analysis
- `Matplotlib` — Data visualization and graphs
- `Scikit-learn` — Machine Learning models
- `TensorFlow` — Deep Learning and Neural Networks

**A Simple ML Example:**
```python
from sklearn.linear_model import LinearRegression
model = LinearRegression()
model.fit(X_train, y_train)
predictions = model.predict(X_test)
```
